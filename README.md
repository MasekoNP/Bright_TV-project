# Bright_TV-project
Subscription growth is a critical success factor in the streaming industry and understanding the user behavior is key to achieving it. This project analyzes BrightTV’s user profile and viewing data to uncover actionable insights that support customer acquisition, engagement and retention strategies. The objective is to leverage data-driven techniques to help the business grow its subscription base and improve overall customer value.

Using SQL and microsoft excel, raw data was transformed into meaningful insights through data cleaning, customer classification and behavioral analysis. The project focuses on identifying high-value users, uncovering conversion opportunities and detecting potential risks of losing customers. By analyzing viewing patterns across different time periods and demographic segments, the project provides a comprehensive understanding of how users interact with the platform.

Key insights from the analysis reveal that viewing activity is higher during weekdays compared to weekends, suggesting optimal timing for targeted campaigns. Additionally, the youth segment emerges as the dominant user group, making it a critical market for growth strategies. Content performance analysis further highlights specific channels that drive higher engagement, emphasizing the importance of content-driven marketing.

The outcomes of this project provide valuable recommendations for the Customer Value Management (CVM) team, enabling them to design more effective marketing strategies, improve customer retention and optimize user targeting. Ultimately, this project demonstrates how data analytics can be used to inform strategic decisions and drive sustainable subscription growth.
